Desafio 1

Densidade demográfica por UF

Cálculo aritmético populacao / area_km2 com ordenação decrescente e classificação automática por faixa de adensamento.

SELECT uf, regiao, ROUND(populacao / area_km2, 2) AS densidade_calculada,
  CASE
    WHEN (populacao / area_km2) >= 100 THEN 'Alta Densidade'
    WHEN (populacao / area_km2) >= 20  THEN 'Média Densidade'
    WHEN (populacao / area_km2) >= 5   THEN 'Baixa Densidade'
    ELSE 'Muito Baixa Densidade'
  END AS classificacao_densidade
FROM censo ORDER BY densidade_calculada DESC;

Desafio 2

Agregação macrorregional — IDH e renda

Médias de IDH e rendimento per capita agrupadas via GROUP BY regiao. Duplo eixo Y para comparação simultânea. Passe o mouse sobre as barras para ver o status classificado automaticamente.

SELECT regiao, COUNT(uf) AS total_estados,
  ROUND(AVG(idh), 3) AS media_idh,
  ROUND(AVG(rendimento_per_capita), 2) AS media_rendimento_per_capita,
  CASE WHEN AVG(idh) >= 0.750 THEN 'IDH Alto' WHEN AVG(idh) >= 0.700 THEN 'IDH Médio-Alto'
    ELSE 'IDH Médio' END AS status_idh,
  CASE WHEN AVG(rendimento_per_capita) >= 2000 THEN 'Renda Alta' ELSE 'Renda Baixa' END AS status_renda
FROM censo GROUP BY regiao ORDER BY media_idh DESC;

Sul, Sudeste e Centro-Oeste formam um bloco homogêneo de IDH Alto e Renda Alta. Norte e Nordeste — 16 dos 27 estados — operam em IDH Médio e Renda Baixa. A renda do Sul (R$ 2.624) é 2,25× maior que a do Nordeste (R$ 1.169). A desigualdade regional brasileira é estrutural, não marginal.

Desafio 3

Frota acima da média nacional

Média calculada via subquery dinâmica — nenhum valor fixo no código. A linha tracejada mostra a média nacional de 4.022.899 veículos calculada em tempo de execução.

SELECT uf, regiao, total_veiculos,
  ROUND((SELECT AVG(total_veiculos) FROM censo), 0) AS media_nacional,
  total_veiculos - (SELECT AVG(total_veiculos) FROM censo) AS excedente_da_media,
  CASE WHEN total_veiculos > (SELECT AVG…) * 3 THEN 'Muito Acima da Média'
    WHEN total_veiculos > (SELECT AVG…) THEN 'Acima da Média' END AS status_frota
FROM censo WHERE total_veiculos > (SELECT AVG(total_veiculos) FROM censo)
ORDER BY total_veiculos DESC;

→ insight analítico · Desafio 3

São Paulo com 35 milhões de veículos é um outlier extremo que distorce a média nacional para cima. Dos 8 estados acima da média, 6 pertencem ao Sudeste e Sul. Os 19 estados abaixo sinalizam dependência crítica de transporte público, que historicamente recebe subinvestimento nas regiões Norte e Nordeste.

Desafio 4

Mapa de vulnerabilidade social

Estados com renda < R$1.500 AND mais de 200k matrículas no ensino fundamental. Eixo X = renda (esquerda = mais crítico). Passe o mouse para ver o estado e seus dados.

SELECT uf, regiao, rendimento_per_capita, matriculas_ensino_fundamental,
  CASE
    WHEN rendimento_per_capita < 1000 THEN 'Vulnerabilidade Crítica'
    WHEN rendimento_per_capita < 1200 THEN 'Alta Vulnerabilidade'
    ELSE 'Vulnerabilidade Moderada'
  END AS nivel_vulnerabilidade,
  CASE WHEN matriculas_ensino_fundamental > 1000000 THEN 'Alta Demanda' ELSE 'Média/Baixa Demanda' END AS demanda_escolar
FROM censo
WHERE rendimento_per_capita < 1500 AND matriculas_ensino_fundamental > 200000
ORDER BY rendimento_per_capita ASC;

→ insight analítico · Desafio 4

12 estados (10 do Nordeste, 2 do Norte) concentram o perfil mais crítico de vulnerabilidade. O Maranhão — R$945 e mais de 1 milhão de matrículas — é o caso extremo: máxima demanda escolar, mínima capacidade de financiamento. Nenhum estado do Sul, Sudeste ou Centro-Oeste atende às duas condições simultaneamente.
